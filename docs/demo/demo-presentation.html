AI Agent — Interactive Configuration

4 Specialized Agents

  • Spend Intelligence — anomalies, duplicates, rates

  • Resource Optimization — licenses, servers

Adjustable thresholds: price anomaly %, duplicate detection window, market rate threshold, max results. Users control what the agent looks for.
